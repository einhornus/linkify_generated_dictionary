больше


I no longer love you.
Я тебя больше не люблю.

I no longer live here.
Я здесь больше не живу.

I can no longer trust him.
Я больше не могу ему доверять.

I can no longer trust them.
Я больше не могу им доверять.

We can no longer be friends.
Мы больше не можем быть друзьями.

Tom can no longer live here.
Том больше не может жить здесь.

уже


You can no longer talk.
Ты уже не можешь говорить.

I no longer remember him.
Я его уже не помню.

Children no longer go to school.
Дети уже не ходят в школу.

They could no longer control themselves.
Они уже не могли себя контролировать.
